# DayFlow — Schedule, Tasks & Points

A beautiful, mobile-first productivity app that gamifies your daily routine. Plan your schedule, track tasks, earn XP, and unlock achievements.

## 🚀 Features

✅ **Daily Schedule** — Plan your day in 30-minute slots (6am–10pm)  
✅ **Quick Tasks** — Add tasks outside your schedule with categories  
✅ **XP System** — Earn points for logging tasks and schedule items  
✅ **Streak Tracking** — Build daily momentum  
✅ **Achievements** — Unlock badges as you progress  
✅ **Level System** — 7 levels from Seedling 🌱 to Legend 🌟  
✅ **Local Storage** — All data saved in your browser, no account needed  
✅ **Push Notifications** — Get reminders for scheduled tasks  
✅ **Mobile Responsive** — Works perfectly on all devices

## 📋 How to Use

1. **Home** — View today's overview, XP progress, and quick task list
2. **Schedule** — Plan tasks in 30-min slots, choose difficulty level (10–50 XP)
3. **Tasks** — Add quick tasks with emoji categories, check them off for 20 XP each
4. **Points** — Track total XP, level progress, achievements, and activity history

## 🎮 How Scoring Works

- **Easy task** (Schedule) = 10 XP
- **Normal task** (Schedule) = 20 XP
- **Hard task** (Schedule) = 35 XP
- **Boss task** (Schedule) = 50 XP
- **Any quick task** (Tasks) = 20 XP

## 🏆 Achievements

- 🌱 **First Step** — Complete your first task
- 🔥 **On Fire** — Get a 3-task streak
- ⭐ **XP Hunter** — Earn 100 XP
- 🏆 **Grinder** — Earn 500 XP
- 📅 **Planner** — Log 5 schedule slots
- ✅ **Task Master** — Complete 10 tasks

### Locally

1. Clone the repo
2. Open `index.html` in your browser
3. Start using it immediately

## 💾 Data Storage

All your data is saved locally in your browser using **localStorage**. Your schedule, tasks, XP, and achievements are never sent to any server. Clear your browser cache and you'll lose your data, so be careful!
